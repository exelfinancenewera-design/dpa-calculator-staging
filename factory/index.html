Learning log — latest

  • Image leads, text follows — hero photo + ≤2 lines above the fold; accordions carry long text.
  • Twin cards equal height AND density — stretch alone leaves dead whitespace.
  • Fixed nav needs scroll-padding — or anchors hide their own headings.
  • Verify Gumroad rendered, never by 200 — "published" can still deliver nothing.
  • Byte-identity is the template gate — extract → build must reproduce the reference exactly.
